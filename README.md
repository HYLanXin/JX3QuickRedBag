# JX3QuickRedBag
一个剑三抢红包的辅助按键，使用winform + Emgu.CV + DD 驱动实现

**PS：建议配合茗伊插件的 聊天监控 使用 可以避免多次重复点击的问题**

## 注意
项目运行需要 **管理员权限** 
> DD驱动需要使用

项目**不支持Debug**调试 请直接使用 **Release** 进行生成 
> DD驱动限制

所有的识图图片存储在项目下**resource**文件夹且 必须根据自己剑三的**文字大小/分辨率** 进行截取的**png文件**

在填完所需参数后需要保存参数，否则无法识别内容
> 未增加数据安全性验证请自行识别 （我是菜鸡没有好的编码习惯

大部分图片验证支持 多图片循环验证 ** 前缀 **在代码里自行寻找

### 自动切换角色 会循环点击输入的坐标 且需要配合茗伊返回角色使用 茗伊返回角色快捷键设置为 数字小键盘的1
